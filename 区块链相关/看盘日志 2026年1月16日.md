    orderedLeaveProb != null ? orderedLeaveProb >= 0.45 : false;

    (trendDir14d === "up" || trendDir14d === "sideways") &&
    (strength14d == null ? true : strength14d >= 40);

  // ================== 4️⃣ 排除 C 类“恐慌一边倒” ==================

  return true;
